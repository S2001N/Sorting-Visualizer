Selection Sort is a widely used Sorting Algorithm in Data Structure & Algorithm.

SELECTION SORT

Selection Sort is the simplest sorting algorithm that finds the minimum element from the array and put that element in the appropriate position.

Algorithm

Step No: 01
Start Algorithm.
Step No: 02
Set minimum (MIN) to location 0.
Step No: 03
Now, Search the minimum element from the remaining list.
Step No: 04
Swap with value at the minimum (MIN) location.
Step No: 05
Increment minimum (MIN) to point to next element.
Step No: 06
Repeat all the steps until the list is sorted.
Step No: 07
End Algorithm.

Know more about Selection Sort

Let's discuss Advantages and Disadvantages of selection sort also discuss the time and space complexity of selection sort.

Advantages

  • Selection Sort is simple and easy to understand and implement.
  • Selection Sort works well with small datasets.

Disadvantages

  • Selection Sort is not a stable sorting algorithm, meaning that elements with the same key value can't maintain their relative order in the sorted output.
  • Selection sort has a time complexity of O(N2) which makes it very slow for large data sets.

complexity

  • Time Complexity: O(N2)
  • Space Complexity: O(1)
