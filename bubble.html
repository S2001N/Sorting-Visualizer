Let's Learn Bubble Sort...

Bubble sort is a Sorting Algorithm, sometimes referred to as sinking sort

BUBBLE SORT

Bubble Sort is the simplest sorting algorithm that works by repeatedly swapping the adjacent elements if they are in the wrong order.

Algorithm

Step No: 01
Start Algorithm.
Step No: 02
Traverse from left and compare adjacent elements and the higher one is placed at right side.
Step No: 03
In this way, the largest element is moved to the rightmost end at first.
Step No: 04
This process is then continued to find the second largest and place it and so on until the data is sorted.
Step No: 05
End Algorithm.

Know more about Bubble Sort

Let's discuss Advantages and Disadvantages of bubble sort also discuss the time and space complexity of bubble sort.

Advantages

  • Bubble sort is easy to understand and implement.
  • It does not require any additional memory space.
  • It is a stable sorting algorithm, meaning that elements with the same key value maintain their relative order in the sorted output.

Disadvantages

  • Bubble sort has a time complexity of O(N2) which makes it very slow for large data sets.
  • Bubble sort is a comparison-based sorting algorithm, which means that it requires a comparison operator to determine the relative order of elements in the input data set. It can limit the efficiency of the algorithm in certain cases.

complexity

  • Time Complexity: O(N2)
  • Space Complexity: O(1)
